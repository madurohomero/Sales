# Sales API 🧾

API de gerenciamento de vendas construída em .NET, com suporte a MongoDB e execução via Docker.

---

## 🚀 Tecnologias

- .NET 8
- MongoDB 7
- Docker
- Docker Compose
- Swagger
- CQRS (MediatR)
- AutoMapper

---

## 📦 Funcionalidades

- Criar venda
- Consultar venda por ID
- Cancelar venda
- Regras de desconto por quantidade
- Persistência em MongoDB

---

# 🐳 Executando com Docker

## ⚠️ Pré-requisitos

Antes de começar, você precisa ter instalado:

- Docker
- Docker Compose
